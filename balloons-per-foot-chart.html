How to use this chart

Find your garland length (measure the real path with string or tape - never eyeball it) and read across for the shopping list. The totals already include roughly 10% extra for pops, tears, and color shortfalls. If you want a count tuned to your exact measurements, fullness, or a different shape - an arch, hoop, column, wall, or number frame - the coverage calculator does the math for any dimensions.

A note on the ranges: decorator styles genuinely vary. A minimalist garland can run as low as 4 balloons per foot; a densely layered organic install can exceed 15. When in doubt, buy toward the higher figure - leftover balloons keep, but a garland that runs short 3 feet from the end doesn't.
